🚀 КАК ЗАПУСТИТЬ ПРОЕКТ:

   ═══════════════════════════════════════════════════════════════════
   ВАРИАНТ A: DOCKER (РЕКОМЕНДУЕТСЯ - 2 МИНУТЫ)
   ═══════════════════════════════════════════════════════════════════

   1. Установите Docker Desktop:
      → https://www.docker.com/products/docker-desktop

   2. Создайте .env файл:
      copy env.docker.example .env       (Windows)
      cp env.docker.example .env         (Linux/macOS)

   3. Отредактируйте .env:
      • Измените POSTGRES_PASSWORD
      • Измените JWT_SECRET_KEY

   4. Запустите:
      docker-start.bat       (Windows)
      ./docker-start.sh      (Linux/macOS)

   5. Откройте:
      http://localhost:5000/swagger

   📖 Подробнее: DOCKER_QUICKSTART.md или DOCKER.md

   ═══════════════════════════════════════════════════════════════════
   ВАРИАНТ B: ЛОКАЛЬНАЯ УСТАНОВКА (5 МИНУТ)
   ═══════════════════════════════════════════════════════════════════

   1. Установите необходимое ПО:
      • .NET 9.0 SDK → https://dotnet.microsoft.com/download/dotnet/9.0
      • PostgreSQL → https://www.postgresql.org/download/

   2. Создайте базу данных:
      psql -U postgres
      CREATE DATABASE renessans;
      \q

   3. Настройте конфигурацию:
      • Откройте: backend/RenessansAPI/appsettings.json
      • Измените: пароль БД и JWT ключ

   4. Проверьте готовность:
      setup-check.bat       (Windows)
      ./setup-check.sh      (Linux/macOS)

   5. Запустите проект:
      start.bat             (Windows)
      ./start.sh            (Linux/macOS)

   6. Откройте в браузере:
      https://localhost:5001/swagger
